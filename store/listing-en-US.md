# ExtWeigh Store listing (en-US)

## Product name

ExtWeigh

## Short description

Find costly Chrome extensions by comparing all-on, all-off, and leave-one-out CPU, Long Tasks, and heap results.

## Description

Chrome feels slow—but is the page responsible, or one of your extensions?

ExtWeigh runs the same browsing scenario with different extension conditions and turns the difference into measurable evidence.

For one extension, it compares enabled and disabled runs. For several extensions, it measures all off, all on, and each leave-one-out condition. The difference between “all on” and “B off” shows the conditional overhead of extension B in your usual combination.

Key features:

- Compare V8 CPU time, Long Tasks, and JavaScript heap
- Measure all-off, all-on, and leave-one-out conditions
- Inspect hot functions from content scripts, service workers, and offscreen documents
- Generate representative URL scenarios from manifest.json
- Use medians, standard deviations, and SIGNIF / NOISE labels across repeated runs
- Review results in the app or a self-contained HTML report
- Keep conditions controlled with disposable Chrome profiles and serial runs

Measurements and reports stay on your local PC. ExtWeigh has no telemetry, advertising, or usage tracking.

Requirements:

- Windows 10 or 11 (x64)
- Google Chrome or Chrome for Testing
- An unpacked Manifest V3 extension folder

Pages that require a signed-in browser session are not supported in the current version.
